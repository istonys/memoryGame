<h3>Memory game!</h3>
<p>A simple memory game that was made using:</p>
<ul>
  <li>HTML / CSS</li>
  <li>JavaScript</li>
  <li>Bootstrap</li>
</ul>
<hr>
<h3>How to play:</h3>
<ol>
  <li>First you are met with a welcoming page. You have to enter your username there.</li>
  <li>From there on all you have to do is press the "Start Game" button.</li>
</ol>
